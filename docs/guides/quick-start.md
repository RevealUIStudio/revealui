# Quick Start

Get a RevealUI app running in 5 minutes.

## Prerequisites

- **Node.js** 24+ ([download](https://nodejs.org))
- **pnpm** 10+ — `npm install -g pnpm`
- A [NeonDB](https://neon.tech) database (free tier works)

## Create Your App

```bash
npx @revealui/cli@latest my-app
cd my-app
```

The CLI prompts for:
- **Template** — `basic-blog` (free) or `e-commerce` / `portfolio` (Pro license required)
- **Database**, **storage**, and **payment** configuration

The CLI writes a `.env.local` file with your answers pre-filled. Any values you skipped will contain placeholder comments.

## Configure Environment

Open `.env.local` and verify at minimum:

```env
# NeonDB connection string (from neon.tech dashboard)
POSTGRES_URL=postgresql://user:password@host/dbname?sslmode=require

# Session signing secret (auto-generated by CLI — do not change)
REVEALUI_SECRET=<generated>

# Admin credentials for the CMS dashboard
REVEALUI_ADMIN_EMAIL=admin@example.com
REVEALUI_ADMIN_PASSWORD=changeme-min-12-chars
```

Change `REVEALUI_ADMIN_EMAIL` and `REVEALUI_ADMIN_PASSWORD` before running in production.

## Run Migrations

```bash
pnpm db:migrate
```

This creates the required tables in your NeonDB database.

## Start the App

```bash
pnpm dev
```

Open [http://localhost:4000/admin](http://localhost:4000/admin) and sign in with the admin credentials you set in `.env.local`.

## Packages Included

| Package | What it does |
|---------|-------------|
| `@revealui/core` | CMS engine, admin UI, collections, rich text |
| `@revealui/auth` | Session auth, sign in/up, rate limiting |
| `@revealui/db` | Drizzle ORM, NeonDB client, migrations |
| `@revealui/presentation` | 50+ UI components, zero external deps |
| `@revealui/config` | Type-safe environment config |
| `@revealui/utils` | Logger, validation helpers |
