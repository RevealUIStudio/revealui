This directory contains auto-generated API documentation for RevealUI Framework packages.

## Packages

- [@revealui/core](../../packages/revealui/README.md) - Core CMS framework

**Note**: API documentation is generated from source code. Individual package README files are located in their respective package directories. Not all packages have README files yet.

## Generation

API documentation is automatically generated from TypeScript source files using JSDoc comments.

To regenerate:

```bash
pnpm docs:generate:api
```

## Note

**Files in this directory are auto-generated. Do not edit them manually.**

Changes should be made to:
1. Source code JSDoc comments
2. TypeScript type definitions
3. The documentation generator scripts
