## Creating Your Own Example

1. **Start with a template:**

```bash
reveal init --template basic
```

2. **Add features incrementally:**

```bash
reveal add payload  # For CMS
reveal add vercel    # For deployment
```

3. **Customize:**

Edit `reveal.config.ts` and add your pages in `src/pages/`.
