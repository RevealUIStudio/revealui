- **React 19** with React Compiler
- **Next.js 16** with App Router
- **TypeScript** with strict mode
- **Triple Database** (REST, Vector, ElectricSQL)
- **OpenAPI 3.2.0** specification
- **Monorepo** with pnpm workspaces
