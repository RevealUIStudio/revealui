# Archive Documentation

**Last Updated**: 2025-01-27

This directory contains archived documentation that is no longer actively maintained but is kept for historical reference.

## Archive Structure

- **`assessments/`** - Historical assessment documents
  - `old/` - Older assessments (pre-2026)
  - `complete-status/` - Outdated "COMPLETE" status files
- **`reports/`** - Historical reports and analyses
- **`migrations/`** - Historical migration guides
- **`implementation/`** - Historical implementation docs
- **`investigations/`** - Historical investigation reports
- **`root-status-files/`** - Historical root-level status files
- **`status/`** - Historical status documents

## Archive Policy

Files are archived when:
1. They are superseded by newer versions
2. They document completed work that is no longer relevant
3. They are duplicates of active documentation
4. They are outdated status reports

## Finding Current Documentation

For current, active documentation, see:
- [Main Documentation Index](../README.md)
- [Current Assessments](../assessments/)
- [Active Guides](../guides/)

## Archive Maintenance

- Archived files are **read-only** - do not update them
- If you need information from an archived file, check if there's a current version first
- Archive is reviewed quarterly for potential cleanup
