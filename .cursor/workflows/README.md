# Cursor IDE Workflows

This folder contains reusable workflows for common development tasks.

## Available Workflows

Workflows define step-by-step processes for completing specific tasks in the RevealUI Framework.

- **analyze-console-errors.md** - Capture and analyze browser console errors using MCP agents (⚠️ Experimental)

## Usage

Reference workflows in Cursor chat:
- "Follow the new collection workflow"
- "Use the component creation workflow"
- "Apply the bug fix workflow"

## Creating Workflows

Create markdown files describing:
- The task or goal
- Step-by-step instructions
- Required files and locations
- Validation steps
- Common pitfalls to avoid
