## Content Creation Tips

### 1. Image Upload Workflow
1. Go to **Media** collection first
2. Upload all images you'll need
3. Note the image names/IDs
4. When creating content entries, select the uploaded images

### 2. Text Content Guidelines
- **Titles**: Keep them short and impactful (3-5 words)
- **Descriptions**: 1-3 sentences, engaging and clear
- **CTAs**: Action-oriented (e.g., "Join Now", "View More")
- **Alt Text**: Descriptive for accessibility

### 3. Content Organization
- Create entries in order of importance
- Use consistent naming conventions
- Add dates for time-sensitive content (events)
- Keep content fresh and updated

### 4. Testing Checklist
- [ ] All images uploaded to Media
- [ ] All images linked in content entries
- [ ] Text content is clear and engaging
- [ ] Links point to correct destinations
- [ ] Frontend displays content correctly
- [ ] Images load properly
- [ ] Mobile view looks good
