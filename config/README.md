# Configuration

Centralized configuration for all RevealUI tools and processes.

## Structure

- `build/` - Build system configuration (turbo, etc.)
- `ci/` - Continuous integration (GitHub Actions, Changesets)
- `ide/` - IDE-specific configurations (VSCode, Cursor, MCP)
- `hooks/` - Git hooks (Husky)
- `docs/` - Documentation system configuration
- `performance/` - Performance budgets and monitoring
- `project/` - Core project configuration
